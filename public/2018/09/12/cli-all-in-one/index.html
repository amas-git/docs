    ANSI escape code

    有时候我们不仅仅需要在终端上输入一些文字, 还有需要控制终端干些别的事儿,比如:

    • 改变光标的位置
    • 改变文字的颜色

    是时候使用转意码大法了:

    当终端读取到这个固定前缀的字符的时候,那么它就意识到后面的是发送给它的命令. 这个想法诞生于1970年.

    我们来看看一个寻常无比的操作:

    1. 在终端上按下右键移动一个字符
    2. 实际上, 终端接受到了一串字符, 在我的电脑上它大概是^[[C
    3. 我们来捕捉一下,输入cat,回车,然后按下方向键下.

    ESC [

    $ print “\e[31mHELLO”

    清空屏幕

    $ print “\ec”

    “\e[0m” : 清楚文字显示属性

    $ print “\e[31mABC\e[0m123”
